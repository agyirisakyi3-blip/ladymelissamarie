## ✨ Features

- **Progressive Web App (PWA)** - Installable on mobile devices with offline support
- **Responsive Design** - Fully optimized for desktop, tablet, and mobile
- **SEO Optimized** - Meta tags, Open Graph, Twitter Cards, structured data (JSON-LD), and XML sitemap
- **Interactive Gallery** - Lightbox-enabled image gallery with hover animations
- **Smooth Animations** - Scroll-triggered animations for engaging user experience
- **Donation Integration** - Multiple giving options (Mobile Money, PayPal, Bank Transfer)
- **Accessibility** - Semantic HTML and proper ARIA attributes

## 🚀 Getting Started

### Prerequisites

- Any modern web browser (Chrome, Firefox, Safari, Edge)
- A local server (for PWA service worker to work properly)

### Running Locally

1. Clone the repository:
   ```bash
   git clone https://github.com/agyirisakyi3-blip/ladymelissamarie.git
   ```

2. Navigate to the project directory:
   ```bash
   cd ladymelissamarie
   ```

3. Serve with any local server. Options:

   **Using Python:**
   ```bash
   python -m http.server 8000
   ```

   **Using Node.js (npx):**
   ```bash
   npx serve
   ```

   **Using VS Code Live Server extension**

4. Open your browser and visit `http://localhost:8000`

## 📱 PWA Installation

The website is a Progressive Web App. To install:

1. Visit the website on a supported browser
2. Look for the install prompt or click "Add to Home Screen" in your browser menu
3. The app will be available on your device and work offline
